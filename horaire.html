Mes cours favoris

Conception Web

Description : Apprentissage des principes de design et des technologies HTML et CSS pour créer des sites web modernes et interactifs.

Ce que j’ai apprécié : J’ai adoré la liberté créative qu’offre la conception web, ainsi que la satisfaction de voir un projet fonctionner sur différents appareils.

Ce que j’ai moins apprécié : Les défis techniques, comme bien positionner une image, étaient parfois frustrants.

Programmation

Description : Ce cours couvre les bases de la programmation en C# .

Ce que j’ai apprécié : La clarté des explications du professeur et les exercices pratiques qui m’ont permis de consolider mes compétences.

Ce que j’ai moins apprécié : Le rythme du cours était parfois rapide, ce qui rendait difficile de suivre certaines notions avancées.
